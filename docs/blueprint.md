# **App Name**: Ledgerly Pro

## Core Features:

- User Authentication: Secure user sign-in/sign-up with Email/Google/Apple via Firebase Authentication.
- Dashboard: Display current balance, total income, and total expenses. Include a '+ Transaction' button for quick entry.
- Transaction Management: Add, edit, and delete income or expense transactions with fields for date, amount, category, and notes. Option to attach receipt images.
- Category Management: Preloaded and custom income/expense categories for detailed tracking.
- Reporting: Generate monthly income vs. expense charts, category breakdown pie charts, and trend lines across time.
- Data Export: Export transaction data in CSV/PDF format for tax purposes.
- Cash Flow Projections: Use a tool to provide AI-driven cash flow projections based on historical data.

## Style Guidelines:

- Background color: Light Emerald, almost white (#F0FAF5)
- Primary color: Deep Emerald (#285943) for trust and sophistication.
- Accent color: Navy (#293A5E) for highlighting key actions and elements.
- Headline font: 'Montserrat' (sans-serif) for headers.
- Body font: 'Inter' (sans-serif) for body text and 'Source Code Pro' (monospace) for numbers.
- Spacious, minimal dashboard layout with emphasis on numbers and charts.
- Subtle micro-animations (e.g., numbers 'counting up' on dashboard load) to signal premium quality.
